CPXXgetenvgroupsize and CPXgetenvgroupsize

This routine accesses the number of environments in a given group of environments.

int CPXXgetenvgroupsize( CPXCENVGROUPptr group )

int CPXgetenvgroupsize( CPXCENVGROUPptr group )

Description

This routine accesses the number of environments in a given group of environments.

Arguments

group
The group to query. If that group is not valid, then the behavior of this routine is undefined.

Return

The number of environments used to create this group.
