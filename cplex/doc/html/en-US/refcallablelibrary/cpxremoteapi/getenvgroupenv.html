CPXXgetenvgroupenv and CPXgetenvgroupenv

This routine accesses an environment from a group of environments.

CPXENVptr CPXXgetenvgroupenv( CPXCENVGROUPptr group, int idx )

CPXENVptr CPXgetenvgroupenv( CPXCENVGROUPptr group, int idx )

Description

This routine accesses an environment from a group of environments.

Arguments

group
The group to query. If the group is invalid, then the behavior of this routine is undefined.
idx
The index of the environment to query. If that index is out of range, this routine returns NULL.

Return

The environment at index idx in this group or NULL if the index is out of range.
