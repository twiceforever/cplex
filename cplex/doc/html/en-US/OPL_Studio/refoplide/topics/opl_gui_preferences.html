IDE Preferences

Describes how to customize the appearance and features of the graphical user interface and the behavior of the editor and other tools, such as CVS, by setting preferences for the IDE.

Choose Window>Preferences in the main menu to open the Preferences dialog box.

To change settings

  1. In the tree structure in the left pane of the Preferences window, click the category of setting you want to change.

    Detailed setting options for that category appear in the right pane of the Preferences window.

  2. Change the settings as required and click Apply.

  3. To revert a changed setting to its default value, click Restore Defaults.

  4. Close the Preferences dialog when you are finished by clicking OK.

To change color settings

  1. Click any color box to display the Color dialog:

  2. The Color dialog appears. Choose a color from the list and click OK.

  3. To define your own colors, click Define Custom Colors and use the additional features to add your own colors to the list.

  4. Close the Color dialog when you are finished by clicking OK.
