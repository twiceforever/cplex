CPXXenvgroupstatus and CPXenvgroupstatus

This routine accesses the status of the most recent previous multicast operation.

int CPXXenvgroupstatus( CPXCENVGROUPptr group, int idx )

int CPXenvgroupstatus( CPXCENVGROUPptr group, int idx )

Description

This routine accesses the status of the most recent previous multicast operation. It returns the status for the most recent previous multicast operation for the environment at index idx in this group.

Arguments

group
The group to query. If that group is not valid, then the behavior of this routine is undefined.
idx
The index of the environment to query. If this value is out of range then the behavior of this routine is undefined.

Return

The status for the most recent previous multicast operation for the specified environment.
