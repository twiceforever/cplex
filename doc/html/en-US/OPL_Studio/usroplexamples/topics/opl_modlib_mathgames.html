Mathematical games

Describes a model that illustrates a didactic mathematical game.

Noughts and crosses

This model describes the three-dimensional Noughts And Crosses (Tic Tac Toe) problem 17 from Model Building in Mathematical Programming, by H.P. Williams.

The characteristics of the noughts and crosses example are the following:

  • Feature: OPL Project
  • Technique: Integer Programming
  • Complexity: Basic
  • Keywords: Mathematical games
  • Project Location: examples/opl/models/NoughtsAndCrosses
  • Model Name: NoughtsAndCrosses.mod
